Admin Sites· 2023

Diseñé y Desarrollé, un dashboard administrativo con roles de usuario, destinado a la gestión eficiente de la estructura y el contenido de sitios web informativos. El propósito principal de este proyecto es tener una solución versátil y ágil, lista para ser implementada en los sitios web de futuros clientes.
